# Spark-Airflow
Dataset management with PySpark and Apache airflow

## Database's with kaggle.com

* [Billboard](https://www.kaggle.com/datasets/jackjaehwankim/billboard-chart-hot-100-oct-2nd)
* [Billboard "The hot 100" Songs](https://www.kaggle.com/datasets/dhruvildave/billboard-the-hot-100-songs)
* [Fifa 20](https://www.kaggle.com/datasets/sagunsh/fifa-20-complete-player-dataset)
* [Top 5000 Albums of All Time](https://www.kaggle.com/datasets/michaelbryantds/top-5000-albums-of-all-time-rateyourmusiccom)
